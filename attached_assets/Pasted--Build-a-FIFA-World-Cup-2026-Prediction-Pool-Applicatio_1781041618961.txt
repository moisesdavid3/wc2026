# Build a FIFA World Cup 2026 Prediction Pool Application

Create a modern, responsive full-stack web application called **World Cup 2026 Predictor Pool**.

## Objective

The application allows multiple users to predict the results of all FIFA World Cup 2026 matches and compete against friends.

Users should be able to:

* Register and log in
* Predict match results
* Save predictions
* View standings
* Compare predictions with actual results
* Earn points based on prediction accuracy
* See who wins the pool

The application should be attractive, mobile-friendly, and easy to use.

---

# Technology Stack

Use:

* Next.js (latest version)
* TypeScript
* Tailwind CSS
* shadcn/ui components
* Prisma ORM

# Automatic Tournament Logic

The system should automatically:

1. Calculate group standings
2. Determine qualified teams
3. Generate knockout bracket
4. Advance winners to next rounds
5. Lock future rounds until previous rounds are completed

Example:

When all group-stage matches have official results:

* Generate Round of 32 automatically

When Round of 32 is completed:

* Generate Round of 16 automatically

And so on until the Final.

# Scoring Rules

Implement:

### Exact score

5 points

Example:

Prediction:
Argentina 2-1 Spain

Actual:
Argentina 2-1 Spain

---

### Correct winner + wrong score

3 points

Example:

Prediction:
Argentina 1-0 Spain

Actual:
Argentina 2-1 Spain

---

### Correct draw

3 points

Example:

Prediction:
1-1

Actual:
0-0

---

### Wrong result

0 points
